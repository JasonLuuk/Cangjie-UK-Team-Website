Classes 

Classes in cangjie are reference objects. This is one of the curcial differences between a class and a struct in cangjieas a struct is a value type. Classes can also inherit from another class while this is not possible for structs.

Classes can be defined as seen here. With a constructor which can be name either init, or the name of the class.

Constructors can be overloaded. For more information check the overloading section of the website.
